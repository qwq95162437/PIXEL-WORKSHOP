我分别在学什么

Unity

游戏原型 / 角色控制 / UI / 关卡逻辑

目前主要通过 Unity 学习 2D / 3D 游戏开发基础， 包括角色移动、碰撞检测、敌人逻辑、物品收集、UI 展示和关卡触发系统。

  • 学习 C# 基础语法与面向对象思路
  • 练习角色控制、跳跃和交互逻辑
  • 尝试将学习内容落地成可玩的 Demo

Unreal Engine

蓝图系统 / 机关交互 / 3D 场景体验

Unreal 目前主要用于探索 3D 解谜和空间交互方向， 我在学习蓝图逻辑、场景搭建、灯光表现以及机关触发流程。

  • 熟悉蓝图事件与交互逻辑
  • 练习关卡氛围与空间节奏设计
  • 测试 3D 解谜原型的可玩性

Blender

建模 / 材质 / 场景资源 / 简单动画

Blender 是我补足资源制作能力的重要工具， 目前重点练习低模建模、材质节点、基础灯光和场景资源整合。

  • 完成简单道具和场景物件建模
  • 练习石材、金属、发光材质效果
  • 为项目原型准备可用资源

Photoshop

像素绘制 / UI 设计 / 宣传图优化

Photoshop 主要用于辅助视觉表达，包括像素图标处理、网页配图优化、 游戏 UI 素材整理与作品展示图制作。

  • 处理网站中使用的图标与头像素材
  • 练习像素风元素与界面排版
  • 优化宣传图与封面展示效果

学习方式

先学基础

先理解软件的核心模块、基础功能和常见工作流，打好使用基础。

再做练习

通过小练习、小模块和功能拆解，把每个知识点单独练熟。

最后做项目

把学到的内容整合进真实页面、原型和作品中，让学习真正产生成果。
