PROJECT DETAIL · INDIE GAME LAB

独立游戏实验室

一个用于记录玩法实验、系统验证和原型迭代的小型项目集合， 目标是把零散想法转化为可测试、可调整、可继续扩展的游戏机制。

系统实验 原型 玩法迭代

项目介绍

独立游戏实验室并不是单一成品项目，而是一个持续展开的小型玩法测试集合。 我会把一些想法拆成最小可验证原型，用来测试它们是否有趣、是否可扩展、是否值得继续做成完整作品。

它更像是一个“实验场”，用于记录不同系统之间的组合关系、交互反馈和玩法节奏。

系统实验

测试不同机制之间是否能形成完整循环，比如战斗、收集或移动系统。

玩法验证

通过最小可玩单元判断一个想法是否足够有趣，是否值得继续打磨。

快速迭代

用更轻量的方式不断试错、调整参数和替换思路，提高开发效率。

开发记录

  • 搭建用于测试系统的基础实验场景
  • 记录多个独立玩法想法与交互草案
  • 尝试不同 UI 反馈与机制组合方式

项目信息

  • 状态：实验中
  • 工具：Unity / Blender
  • 类型：玩法实验 / 原型测试
  • 方向：系统原型 / 机制验证
  • 负责：机制设计 / UI / 实验记录
