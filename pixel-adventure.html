PROJECT DETAIL · PIXEL PLATFORMER

像素冒险 Demo

一个复古像素风的平台跳跃原型项目，围绕关卡探索、跳跃反馈、 敌人 AI 与收集系统展开开发。

平台跳跃 像素风 关卡探索

项目介绍

这个项目是我在学习 Unity 与 C# 过程中做的一个像素平台跳跃 Demo。 目标是通过一个可玩的原型，把角色移动、跳跃、敌人逻辑、碰撞检测、 金币收集和关卡节奏这些基础模块整合起来。

它不仅是一个练习项目，也是在尝试把“学习的软件”真正做成“看得见的作品”。

角色控制

实现基础移动、跳跃与落地反馈，提升操作手感。

关卡设计

加入平台落差、收集路线与探索空间，形成基础节奏。

敌人与收集

加入敌人 AI 与金币系统，让关卡不只是移动练习。

开发记录

  • 完成角色移动、跳跃与碰撞基础逻辑
  • 加入金币收集与简单敌人巡逻系统
  • 迭代第 2 关，增加移动平台机制

项目信息

  • 状态：持续迭代中
  • 引擎：Unity
  • 语言：C#
  • 类型：2D 平台跳跃
  • 负责：程序 / UI / 关卡
